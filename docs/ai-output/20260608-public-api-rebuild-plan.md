# 마곡동 상업용 부동산 월단위 API 재구성 계획

## 목표

마곡동 개별 건물별 10년 상업용 부동산 거래를 월단위로 재구성하고, CSV에 없는 건물명/층/용도/면적 정보를 승인된 공공데이터 API로 보강한다.

## 활용신청 API 분류

### Primary

- 국토교통부_상업업무용 부동산 매매 실거래가 자료
  - 월단위 거래 원천으로 사용한다.
  - 법정동코드, 계약년월, 지역 범위로 반복 호출한다.
- 국토교통부_건축HUB_건축물대장정보 서비스
  - 대지위치, 지번, 도로명, 건물명, 주용도, 층수, 연면적 보강에 사용한다.

### Support

- 국토교통부_오피스텔 매매 실거래가 자료
- 국토교통부_오피스텔 전월세 실거래가 자료
- 국토교통부_토지 매매 실거래가 자료
- 국토교통부_공장 및 창고 등 부동산 매매 실거래가 자료
- 전국공인중개사사무소표준데이터

### Context

- 아파트/연립다세대/단독다가구 매매 및 전월세 API는 상업용 직접 원천이 아니라 주변 시장 맥락으로만 사용한다.

## 인증키 처리

- 인증키는 저장소 파일에 저장하지 않는다.
- 실행 시 `PUBLIC_DATA_SERVICE_KEY` 환경변수에서만 읽는다.
- 화면/답변에 인증키를 출력하지 않는다.

## 월단위 재구성 방식

1. `2017-01`부터 현재월까지 월 목록을 생성한다.
2. 상업업무용 매매 실거래 API를 월별 호출한다.
3. 마곡동 법정동코드 또는 시군구/법정동 필터로 정제한다.
4. 지번/도로명/면적/층/용도 기반으로 건물 후보 키를 만든다.
5. 건축물대장 API로 건물명과 건축물 정보를 보강한다.
6. 월별 평균 거래금액, 중위 거래금액, 거래건수, 전용평당가를 집계한다.
7. CSV와 API 결과를 비교해 누락월/누락건/마스킹 차이를 리포트한다.

## 현재 반영 상태

- CSV 기반 월단위 그래프는 대시보드에 반영 완료.
- API 목록은 `data/processed/approved-api-inventory.json`에 저장 완료.
- API 호출은 인증키 연결 후 실행 가능하도록 다음 단계에서 추가한다.
